# Demand Forecasting & Inventory Decision Analysis

## Objective
Forecast weekly demand for a retail store and simulate inventory decisions to reduce stockouts and overstock.

## Dataset
Walmart Store Sales Dataset (single-store time series)

## Methods
- Time series EDA (trend, seasonality, volatility)
- Forecasting using Facebook Prophet
- Inventory planning using forecast uncertainty (upper/lower bounds)

## Key Insights
- Strong weekly seasonality observed
- Demand spikes during holiday periods
- Forecast uncertainty used to estimate safety stock

## Tools
Python, Pandas, Matplotlib, Prophet
